# Expense Tracker

A personal expense tracker built with Python.

## Features

- Add expenses
- View expenses
- Delete expenses
- Search by category
- Monthly summary
- Category summary
- Input validation

## Technologies Used

- Python
- File Handling
- Dictionaries
- Exception Handling

## How to Run

1. Open the project
2. Run `main.py`
3. Use the menu options
